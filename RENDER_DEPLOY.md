# 외부 접속 배포 체크리스트

이 프로젝트는 로컬 실행뿐 아니라 Render/Replit 같은 클라우드 서비스에 올려 외부 접속이 가능하도록 구성했다.

## 배포 가능하게 만든 코드 요소

- `app.run(host="0.0.0.0", port=os.environ.get("PORT", 5000))`
  - 로컬 컴퓨터 내부 주소에만 묶이지 않고 외부 요청을 받을 수 있게 한다.
- `requirements.txt`
  - Flask와 gunicorn 설치 목록을 명시한다.
- `Procfile`
  - Render가 어떤 명령으로 서버를 켜야 하는지 알려준다.
- `runtime.txt`
  - Python 버전을 고정해 배포 환경 차이로 생기는 오류를 줄인다.
- `/health`
  - 서비스가 정상 작동하는지 간단히 확인할 수 있는 주소다.
- `/api/stats.json`
  - 웹 화면뿐 아니라 데이터 API 형태로도 분석 결과를 확인할 수 있다.

## 제출 직전 확인할 것

- 공개 URL 접속 가능 여부
- 새 점수 입력 가능 여부
- 곡별 상세 페이지 이동 가능 여부
- CSV 다운로드 가능 여부
- 모바일 화면에서 레이아웃이 깨지지 않는지
